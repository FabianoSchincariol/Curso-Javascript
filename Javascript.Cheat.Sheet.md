### Comentários em Javascript

Comentário de uma única linha com duas barras ->   //

Comentários de multiplas linhas com barra asteristico e asteristico barra ->  /**/

```Javascript

// cometário de única linha

/*
    Comentário de multiplas linhas
*/

```
